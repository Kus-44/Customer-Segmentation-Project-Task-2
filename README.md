# Customer Segmentation Analysis using Power BI & Excel

## Project Overview

This project analyzes customer data to identify different customer groups based on their behavior and demographics. The main aim is to understand customer patterns and support better business decisions using data visualization.

## Objective

* Segment customers based on income, spending score, and frequency
* Understand customer behavior patterns
* Create an interactive dashboard for insights

## Tools Used

* Microsoft Excel
* Power BI

## Dataset Description

The dataset contains customer-related information such as:

* Customer ID
* Age
* Gender
* Annual Income
* Spending Score
* Purchase Frequency

## Data Preparation

* Removed missing and duplicate values
* Structured data in Excel for analysis
* Created calculated fields where required

## Dashboard Features

* Total Customers, Average Income, Average Spending Score (KPI Cards)
* Gender distribution using Pie Chart
* Income vs Spending Score using Scatter Plot
* Customer segmentation using clustered analysis
* Interactive filters for better exploration

## Key Insights

* Customers can be grouped into distinct segments based on spending behavior
* High income does not always mean high spending
* Some customer groups contribute more to overall business value
* Helps in identifying target customers for marketing campaigns

## Conclusion

The dashboard provides clear insights into customer behavior and helps businesses make data-driven decisions for improving marketing and customer targeting strategies.
