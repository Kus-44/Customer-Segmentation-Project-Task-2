📊 Customer Segmentation Analysis using Power BI & Excel
🧾 Project Overview

This project focuses on analyzing customer data and grouping customers into meaningful segments based on their behavior and demographics. The goal is to help businesses understand their customers better and support data-driven decision-making for marketing and sales strategies.

🎯 Objective
Identify different types of customers based on income, spending behavior, and frequency
Discover hidden patterns in customer data
Build an interactive dashboard for business insights
🛠️ Tools & Technologies Used
Microsoft Excel – Data cleaning and preparation
Power BI – Data visualization and dashboard creation
📁 Dataset

The dataset includes customer details such as:

Customer ID
Age
Gender
Annual Income
Spending Score
Purchase Frequency
📊 Dashboard Features

The Power BI dashboard includes:

📌 KPI Cards:
Total Customers
Average Income
Average Spending Score
Average Purchase Frequency
📊 Visualizations:
Pie Chart – Gender distribution
Scatter Plot – Income vs Spending Score (Customer segmentation)
Clustered/Bar Charts – Customer group analysis
Segmentation view based on behavior patterns
📈 Key Insights
Customers are divided into different segments based on spending behavior
High-income customers are not always high spenders
A specific group of customers contributes most to revenue
Clear patterns help in identifying target marketing groups
🚀 Outcome

An interactive Power BI dashboard that transforms raw customer data into actionable insights for business decision-making and customer targeting strategies.

📌 How to Use
Open the Excel file to view raw data
Open the Power BI file (.pbix)
Explore interactive dashboards and filters
Analyze customer segments and insights
